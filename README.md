## Prerequisites
- python (i recommend 3.6.1 or higher )
- pygame 1.9.4 or higher :https://www.pygame.org/wiki/GettingStarted
- any python interpretor

## how to use
1. download everything into a file
2. open Stack.py and save it into the folder 
3. run Stack.py
